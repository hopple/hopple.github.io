Building Static Blog Using Pelican and Github

on 2013 November 20 in Tutorial about python . github . pelican

This tutorial describes how to build a static blog using pelican and github. Pelican is a python based static content generator like jekyll which can transform the plain text into static websites and blogs. Github provides static content hosting which can be used to host your static blog for free. I will talk about the idea behind the Pelican and describe the main process of building a static blog with Pelican.
